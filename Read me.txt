
*Run the Script:
Download smb python script.
Open a Command Prompt.
Put the directory of the input pcap file (i.e /path/to/input.pcap) and output file path (i.e /path/to/output.json) next to script. 
Run the script using Python:

    > python smb.py /path/to/input.pcap /path/to/output.json
